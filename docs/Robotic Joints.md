# Robotic Joints
- Joints connect the [[Manipulator|manipulator]] links
- A joint normally provides one Controllable Degree of Freedom (CDOF)
- For each CDOF one separate [[Actuator|actuator]] is needed
- An endeffector with many (C)DOFs needs a lot of actuators!
- [[Rotary Joint]] , [[Prismatic Joint]]

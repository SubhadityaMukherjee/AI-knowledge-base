Response Based Knowledge¶

  • neural response of the last output layer of the teacher model. The main idea is to directly mimic the final prediction of the teacher model.
  • Given a vector of logits z as the outputs of the last fully connected layer of a deep model, the Distillation Loss for response-based knowledge can be formulated as
  • response in object detection task may contain the logits together with the offset of a bounding box (Chen et al., 2017). In semantic landmark localization tasks, e.g., human pose estimation, the response of the teacher model may include a heatmap for each landmark (Zhang et al., 2019a)

# Tau Protein
- A type of protein abundantly found in neurons. When this protein is not adequately cleared from the brain, it can form tangles that are a key pathology of several neurodegenerative disorders including frontotemporal degeneration, CTE, and [Alzheimer’s disease](Alzheimer’s%20Disease.md).

# Bayesian Information Criterion
- $$BIC = -2 log(L) + 2log(n)q = \frac{1}{n}(RSS + log(n) p \hat \sigma^{2})$$
- L: denotes the likelihood function for a particular model
- q: number of estimated parameters of the model

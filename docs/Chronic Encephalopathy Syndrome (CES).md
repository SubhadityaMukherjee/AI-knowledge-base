# Chronic Encephalopathy Syndrome (CES)
- Symptoms, including memory issues, depression, and impulsive behavior, that manifest themselves after repeated brain traumas. Over time, CES can result in a diagnosis of [chronic traumatic encephalopathy (CTE)](Chronic%20Traumatic%20Encephalopathy%20(CTE).md).

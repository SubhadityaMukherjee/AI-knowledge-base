# Safety Logic Circuit
- The safety logic circuit monitors safety critical external devices such as the light curtains and FSU generated signals. The safety logic circuit is programmed via an intuitive user interface that is supported on the Yaskawa programming pendant. It enables to set up the logical operations, such as stopping the [manipulator](Manipulator.md) or outputting a signal if the servos are on.

# Sparse Dict Learning Loss
- $$L(X) = n^{-1}\Sigma_i^n ||x_i - Dr_i ||^2 + \lambda \Sigma_i |r_i|$$
- $\lambda \Sigma_i |r_i|$ is Lasso/L1 [[Lp Regularization]]
- Predictions :  $$r = argmin_r ||x- Dr_i ||^2 + \lambda \Sigma_i |r_i|$$

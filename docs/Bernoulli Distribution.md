# Bernoulli Distribution
- Only two possible outcomes
- [[PMF]] : $$p(s_{i}) = \begin{cases} 1-q, & \text{for i = 1} \\ q,& \text{for i =2}  \end{cases}$$
- Given : Data - ${x_{1}, .., x_{N}}$ and $x_{i} \in {s_{1}, s_{2}}$ then $$\hat q = \frac{1}{N}|\{i|x_{i}= s_{2}\}$$

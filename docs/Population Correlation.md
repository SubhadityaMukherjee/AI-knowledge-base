# Population [[Correlation]]
- $$\rho_{xy}= \frac{\sigma_{xy}}{\sigma_{x}\sigma_{y}}$$
- $\sigma$ is the [[Standard Deviation]]
- $\sigma_{xy}$ is the [[Covariance]]

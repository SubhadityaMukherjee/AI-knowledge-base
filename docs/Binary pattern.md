# Binary Pattern Encoding
- If symbol alphabet has a large size k
	- [[One hot]] is too huge
- Encode into binary vector of length $$\lceil log_{2} \rceil$$
- {a,b,c,d} -> {[0,0]', [0,1]', [1,0]', [1,1]'}
- Non linear effort as it is a arbitrary encoding
- Too intensive

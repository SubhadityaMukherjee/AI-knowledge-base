# Multinomial
- $$P(D|\theta) = \frac{N!}{n_{1}!…n_{l}!}\Pi_{j=1}^{l}\theta_{j}^{n_{j}}$$
- $N = n_{1}+ ….+ n_{l}$
- Generalized [[Binomial Distribution]]

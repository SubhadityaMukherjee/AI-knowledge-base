# Probability Mass Function
- pmf
- Given a discrete sample space S
	- S is a function $p : S \rightarrow [0,1]$ whos total mass is 1
	- satisfies $\Sigma_{s \in S}p(s) = 1$## Backlinks
* [[Poisson Distribution]]
	* [[PMF]] : $$p(k) = \\frac{\\lambda^{k}e^{-k}}{k!}$$
* [[Binomial Distribution]]
	* [[PMF]]: $$p(s) = \\binom{N}{s}q^{s}(1-q)^{N-s} = \\frac{N!}{s!(N-s)}q^{s}(1-q)^{N-s}$$
* [[Bernoulli Distribution]]
	* [[PMF]] : $$p(s\_{i}) = \\begin{cases} 1-q, & \\text{for i = 1} \\ q,& \\text{for i =2}  \\end{cases}$$

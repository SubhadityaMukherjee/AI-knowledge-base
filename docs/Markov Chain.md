# Markov Chain
- Sequence of random variables such as $X_{n+1}$ only depends on $X_{n}$
- Discrete time
- Stochastic process without memory
- Finite interval : [0, 1, … n]
- Right infinite : n = 1, 2, 3, …
- Left right infinite : Integers
	- No start

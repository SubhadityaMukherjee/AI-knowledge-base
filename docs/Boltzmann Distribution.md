# Boltzmann Distribution
- [[PDF]] $$p(s|T)= \frac{1}{\int_{s}e^{-C(s)/T}ds}e^{-C(s)/T}$$
- Energy function $E: S \rightarrow \mathbb{R}^{\geq 0}$

# Random Directions


- 1D
	- $$
f(\alpha, \beta) = L(\theta^{*}+ \alpha \delta)
$$
- 2D
	- $$
f(\alpha, \beta) = L(\theta^{*}+ \alpha \delta + \beta \eta)
$$
- $\theta, \eta$ are random direction vector and $\theta^{*}$ is optimal weights vector
